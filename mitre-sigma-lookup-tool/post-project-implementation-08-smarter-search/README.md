## Post Project Implmentation #8: Smarter Search
- This implementation focused on improving search by supporting partial and approximate matches for names and descriptions, returning relevant techniques for users to choose from. I also added a pagination feature where it has a "Next Page" option if there are more than 25 matches and gives the user the choice to type `n` to see the next batch of 25 and `b` to go back to the last page
