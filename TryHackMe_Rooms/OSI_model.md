# TryHackMe - OSI Model Writeup

## Overview
This room explains the **OSI Model**, a framework describing how data moves across networks.

---

## Step 1: OSI Model Layers

**Layer 1 – Physical:** hardware, cables, signals  
**Layer 2 – Data Link:** MAC addressing, frame transmission  
**Layer 3 – Network:** IP addressing, routing (OSPF, RIP)  
**Layer 4 – Transport:** TCP (reliable, handshake), UDP (fast, connectionless)  
**Layer 5 – Session:** manages device communication sessions  
**Layer 6 – Presentation:** translates/encrypts data (HTTPS)  
**Layer 7 – Application:** network services used by apps (HTTP, DNS, FTP, SMTP)

---

## Key Learning Points
- The OSI Model standardizes network communication.  
- Each layer has a specific role, from physical transmission to application services.  
- Understanding layers helps troubleshoot and secure networks.
